# Sumit Kumar's Portfolio

A modern, professional portfolio website built with React, showcasing skills, projects, certifications, and achievements.

## Features

✨ **Fully Responsive Design** - Mobile, tablet, and desktop optimized
🎨 **Modern UI/UX** - Beautiful gradient designs and smooth animations
📱 **Interactive Components** - Smooth scrolling and hover effects
🖼️ **Certificate Gallery** - Upload and display certificate images
📄 **CV Download** - One-click resume download
📧 **Contact Form** - Get in touch directly
🔗 **Social Integration** - Links to GitHub, LinkedIn, and email

## Sections

- **Home** - Hero section with quick intro
- **About** - Personal summary and statistics
- **Skills** - Technical skills with proficiency bars
- **Projects** - Featured projects with descriptions
- **Certifications** - Certificates with image upload capability
- **Achievements** - Key milestones and accomplishments
- **Education** - Academic background and timeline
- **Contact** - Contact form and social links

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **React Icons** - Icon library
- **React Scroll** - Smooth scrolling

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

The portfolio will open at `http://localhost:3000`

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `dist` folder.

## Customization

### Update Personal Information
Edit the components to add your own information:
- Contact details
- Skills and experience
- Projects and achievements
- Education background

### Modify Styling
- Update colors in `tailwind.config.js`
- Modify CSS animations in `src/index.css`
- Change component styles in individual component files

### Add Certificate Images
In the Certifications section, click on any certificate box to upload an image.

## Deployment

The portfolio can be easily deployed to:
- Vercel (recommended for Vite projects)
- Netlify
- GitHub Pages
- Firebase Hosting

### Deploy to Vercel

```bash
npm install -g vercel
vercel
```
